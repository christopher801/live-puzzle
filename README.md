# 📸 Live Camera Puzzle

A modern web application that captures your live webcam image, splits it into puzzle pieces, and lets you solve it via drag and drop. Built with React, Vite, Bootstrap, and dnd-kit.

![Live Camera Puzzle](public/favicon.svg)

## ✨ Features

- 📷 **Live Camera Capture**: Take a photo using your device's camera
- 🧩 **Dynamic Puzzle Generation**: Split your photo into 3x3, 4x4, or 5x5 puzzle pieces
- 🖱️ **Drag & Drop**: Rearrange pieces smoothly using dnd-kit
- ⏱️ **Timer**: Track how long it takes to solve the puzzle
- 🎉 **Win Detection**: Automatic celebration when puzzle is solved
- 📱 **Mobile Responsive**: Works perfectly on all devices
- 🎨 **Beautiful UI**: Bootstrap 5 with custom animations
- 🔄 **Camera Switch**: Toggle between front and back cameras


## 🛠️ Technologies Used

- **React 18** - UI library
- **Vite** - Build tool and development server
- **Bootstrap 5** - Styling and responsive design
- **dnd-kit** - Drag and drop functionality
- **react-webcam** - Camera access and capture
- **Framer Motion** - Smooth animations
- **Canvas API** - Image processing and splitting

## 📋 Prerequisites

Before you begin, ensure you have installed:
- Node.js (v18 or higher)
- npm or yarn
- A modern web browser with camera access (Chrome, Firefox, Safari, Edge)
